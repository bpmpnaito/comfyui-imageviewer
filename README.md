# ComfyUI Image Viewer

Simple image viewer node for ComfyUI.

## Installation

1. Copy this folder to `ComfyUI/custom_nodes/comfyui-imageviewer`
2. Restart ComfyUI

## Usage

Add the "Image Viewer" node from the image category and connect your images to it.

## Files

- `nodes.py` - Python node definition
- `web/imageviewer.js` - JavaScript extension
- `__init__.py` - ComfyUI registration
